Что такое IP домофон?

Современное высокотехнологичное устройство

Беспроводной ip домофон – это вызывная информативная панель, которая снабжена IP камерой, микрофоном с динамиком, клавиатурой для набора или функциональными кнопками для вызова. Панель подключена стандартной витой парой к локальной сети, через которую осуществляется выход в интернет.
